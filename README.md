# Lileo Codebuddy-code

## How do I install these formulae?

`brew install lileo/codebuddy-code/<formula>`

Or `brew tap lileo/codebuddy-code` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lileo/codebuddy-code"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
